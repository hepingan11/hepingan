下面是我自己手写的：（用于自己学习记录用的）

strong 加粗 b
ins 下划 u
em 倾斜 i(italic)
del 删除线 s
br 换行 hr 水平线 p自动换行
src路径 controls控制 loop循环 autoplay自动播放(mutec视频静音播放)
# 空连接 href超链接跳转 target(_self覆盖原网页 _blank新窗口打开)
dl自定义列表 dt自定义列表主题 dd自定义列表的内容 ul无序列表 ol有序列表 li每一项
table表格 tr表格行 td单元 border边框 rowspan合并垂直 colspan合并水平(左上原则)
input:输入 text单行文本 password输入密码 radio单选框(用于多选一) checkbox多选框,用于多选多
file上传文件 submit提交按钮 reset重置按钮 button普通按钮(默认无功能) placeholder占位符，提示输入内容
checked默认选中 multiple选中多个文件(file上传文件)
select下拉菜单 option下拉每一项
textarea文本域标签
label绑定内容欲表单标签的关系
div背景图标签 spone span行内标签
header网站头部 nav 网站导航 footer网页底部 aside侧边栏 section区块 article文章
style css核心 color颜色 font-size字体大小 background-color背景颜色(width height)
font:直接输入参数就行 text-decoration:字体设置
background-image: url()：背景图 “>display flex
margin调整位置
border-bottom下划线
